SCSS Guide

Change Colors

// assets/scss/_variables.scss
:root {
    --primary-500: #6366F1;      // Main color
    --accent-500: #8B5CF6;       // Accent color
}

[data-theme="light"] {
    --bg-primary: #ffffff;
    --text-primary: #0a0a0a;
}

Compile SCSS

# Using VS Code Live Sass Compiler
1. Install "Live Sass Compiler"
2. Click "Watch Sass"

# Command line
cd PortSpace
sass --watch assets/scss/main.scss:assets/css/main.css

Variables

Variable Purpose Default
--primary-500 Main color #6366F1
--accent-500 Accent color #8B5CF6
--font-primary Body font 'Inter', sans-serif
--font-secondary Heading font 'Clash Display', sans-serif

Dark/Light Mode

<html data-theme="dark">  <!-- Default -->
<html data-theme="light"> <!-- Light mode -->

// assets/js/app.js
const themeToggle = document.getElementById('themeToggle');
themeToggle.addEventListener('click', () => {
    const html = document.documentElement;
    const newTheme = html.dataset.theme === 'dark' ? 'light' : 'dark';
    html.setAttribute('data-theme', newTheme);
    localStorage.setItem('theme', newTheme);
});
